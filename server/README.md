Formatting Code
### `npx prettier --write **/*.js`
Run Server
### `nodemon index.js`

Install Package
### `npm install`

Install cli
### `npm install --save-dev sequelize-cli `

DB
### `npx sequelize-cli db:create`
### `npx sequelize-cli db:migrate`

SEED
### `npx sequelize-cli db:seed:all`
### `npx sequelize-cli seed:generate --name demo-user`
